# UK Crime Analysis Dashboard
<br>
1. Project Overview<br>
This project analyses UK crime data from January 2025 to August 2025 using an interactive
Tableau dashboard. The objective is to identify crime patterns, high-risk locations, and trends over
time to support data-driven decision-making in public safety and resource allocation.<br>
2. Business Problem<br>
Local authorities require insights into crime distribution and trends to improve policing strategies,
reduce crime rates, and optimise deployment of resources.<br>
3. Data & Tools<br>
Data Source: UK Police public datasets
Tools Used: Power BI <br>
4. Data Preparation <br>
 Removed duplicate records and standardised location data<br>
 Handled missing values in crime outcome and location fields<br>
 Extracted time-based features such as month and day<br>
 Prepared dataset for visualisation in Tableau<br>
5. Dashboard Features<br>
Crime distribution by type<br>
Monthly crime trends<br>
Location-based crime analysis<br>
Interactive filters for crime type, date, and location<br>
6. Key Insights
 Theft-related crimes are the most common, with 'Other Theft' and 'Shoplifting' leading.<br>
  Violence and public order offences form a significant share of total crimes.<br>
 Crime trends rise from January to May 2025 before slightly declining.<br>
 A small number of locations account for a high proportion of crimes.<br>
 Bicycle theft is a frequent issue in central areas.<br>
 Lower-frequency crimes remain consistent but less impactful overall.<br>
 Recommendations<br>
1 Focus policing efforts on high-crime locations<br>
2 Implement targeted strategies for theft-related crimes<br>
3 Use trend analysis for proactive planning<br>
4 Monitor seasonal crime variations<br>
